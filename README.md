# TP_10_NODE
This project is meant to store TP10 node.js for INTES IMT LILLE DOUAI.

To make it run :

1) create passwordDB.js in /config
2) paste this:

module.exports = {
    PASSWORD: <ur_password>,
};

replace <ur_password> by ur MySQL root password

3) Open terminal in root folder 
4) npm intall
5) npm run start
6) Enjoy :)
